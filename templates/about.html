About This Project

This project is designed to classify handwritten digits using a convolutional neural network (CNN) model. You can upload an image of a digit, and the model will predict the digit based on the input.

Technologies Used

  • Python (Flask)
  • TensorFlow & Keras
  • HTML, CSS
